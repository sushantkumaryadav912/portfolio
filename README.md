# Sushant Kumar Yadav - Professional Developer Portfolio

A stunning, high-performance single-page developer portfolio website built using standard semantic HTML5, custom CSS3 (glassmorphism/dark theme), and Vanilla JavaScript. Built based on the professional resume of Sushant Kumar Yadav.

## 🚀 Live Demo & How to Run

Since the portfolio is built using pure, native web technologies (HTML, CSS, JS), you do not need any compilation step.

### Option 1: Direct Open
Simply double-click the `index.html` file or open it directly in any modern web browser.

### Option 2: Live Local Server
For development with hot-reloading:
```bash
# Python 3
python3 -m http.server 8000

# Node.js
npx serve .
```
Then visit `http://localhost:8000` or the port displayed in your terminal.

## 🎨 Design & Aesthetic Features

- **Dark Operative Mode**: Deep midnight background (`#0a0f1d`) with custom radial gradients for ambient lighting effects.
- **Glassmorphism Components**: Card elements use subtle semi-transparent white fills, backdrop blurs, and border highlights mimicking frosted glass.
- **Typewriter Effect**: Hero header animations rotating through core focus areas.
- **Scroll-Triggered Reveals**: Dynamic section and element reveals using high-performance JavaScript `IntersectionObserver`.
- **Responsive Layout**: Fluid breakpoints optimized for mobile, tablet, and widescreen monitors.
- **Micro-interactions**: Scale transforms, custom glow effects, and transitions on cards, buttons, badges, and contact options.

## 📂 Project Structure

- `index.html`: Contains all 8 semantic sections (Hero, About, Experience, Projects, Skills, Certifications, Achievements, Contact) pre-filled with up-to-date resume details.
- `style.css`: Houses CSS custom variables (theming system), component layouts, animations, transitions, and media queries.
- `script.js`: Handles interactive mechanics (typewriter script, scroll animations, mobile hamburger menu toggler, contact form mockup response).
- `SushantResume_Updated.pdf`: Original resume source document.
